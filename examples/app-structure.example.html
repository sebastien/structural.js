Structural Model

Structural support the following types:

text
Plain text content, you can edit it as you want.
skip
Plain text can contain skip elements that are not editable nor selectable.
atom
Atomic element atom that is not editable but selectable.
container
Container elements, like a a placeholder are selectable and have text inside that can be edited. The cursor entering/exiting will first select the placeholder before proceeding to the next text.
templates
Templates are like containers however their content is skipped unless there's a slot defined. For instance template:slot.
