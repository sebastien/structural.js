Rich Text Editor

This is a rich text editor built with Structural.js. Try formatting text using the toolbar above or keyboard shortcuts.

Select some text and toggle bold, italic, or monospace inline styles. You can also click a word to format it without selecting.

Block Formatting

Use the heading buttons to promote paragraphs into heading levels. Create bullet lists for structured content.

Blockquotes are great for highlighting quotes, citations, or callout content. Toggle the button again to remove.

Try It Out

Place your cursor anywhere and click the toolbar buttons. The active format is highlighted in blue. Keyboard shortcuts work too: Ctrl+B for bold, Ctrl+I for italic, Ctrl+\` for monospace, and Ctrl+1/2/3 for headings.
